# docker-nettools

A simple, small, debian-based Docker image with some handy networking tools installed:

- curl
- hping3
- nc
- nping
- nslookup
- dig
- tcpdump
- socat
- telnet
- jq

Please see the [Dockerfile](https://github.com/dyrnq/docker-nettools/blob/master/Dockerfile) for a complete list of tools.
